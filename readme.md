c_implementation.of->{space_invaders};

## running

```sh
./output/space_invaders
```

```sh
make run
```

### parameters

- `--skip-intro` : skips the intro screen
- `--fill-screen`: fills the entire screen regardless of size

## dependencies

- [alic3dev](https://github.com/alic3dev)
- - [`cexil`](https://github.com/alic3dev/cexil)
- - [`clic3`](https://github.com/alic3dev/clic3)
- - [`interrupt_handler`](https://github.com/alic3dev/interrupt_handler)

make
